🛠️核心服务🛠️

华中科技大学开源镜像站

镜像站目前已同步多项开源软件镜像，有效镜像数据超过40T，且支持公网访问

Gitea 代码托管平台

支持代码管理、问题跟踪和团队协作的代码托管服务，基于 HUST 身份认证

Seafile 文件存储平台

提供文件存储、同步和共享协作的云存储服务，基于 HUST 身份认证

在线 LaTeX 协作平台

支持编译环境和实时协同编辑的在线 LaTeX 写作服务，基于 HUST 身份认证

pwn.hust.college 教育实践平台

趣味安全教育实践平台，学习并实践网络空间安全核心攻防技术，基于 HUST 身份认证

Linux 内核高校贡献排行榜

基于 Linux 内核 Git 仓库数据，统计各高校（按域名）的开源贡献

开源活动截止日期项目

一站式展示开源会议、竞赛和活动截止日期，助力社区参与和贡献

RISCV Online

基于 WebAssembly 的 RISC-V 汇编反汇编工具，无需安装即可在浏览器中使用
