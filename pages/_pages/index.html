🛠️核心服务🛠️

华中科技大学开源镜像站

镜像站目前已同步多项开源软件镜像，有效镜像数据超过40T，且支持公网访问

Gitea 代码托管平台

基于 HUST 身份认证的 Gitea 代码托管服务，支持代码管理、问题跟踪和团队协作

Seafile 文件存储平台

基于 HUST 身份认证的 Seafile 文件存储服务，提供文件存储、同步和协作

在线 LaTeX 协作平台

基于 HUST 身份认证的在线 LaTeX 协作服务，提供编译环境和实时协同编辑等功能

pwn.hust.college 教育实践平台

基于 HUST 身份认证的趣味安全教育实践平台，学习并实践网安核心攻防技术

Linux 内核高校贡献排行榜

基于 Linux 内核 Git 仓库数据，统计各高校（按域名）的开源贡献

开源活动截止日期项目

一站式展示开源会议、竞赛和活动截止日期，助力社区参与和贡献

riscv-online

基于 WebAssembly 的在线 RISC-V 汇编反汇编工具

⭐明星项目⭐

🖥️ HUST Mirror

华中科技大学开源镜像站

🗓️ Open Source Event Deadlines

开源活动、会议和竞赛一站式展示

🌐 HCTT 翻译团队

HCTT(HUST openatom Club Translation Team) 是“华科开放原子开源俱乐部”的翻译团队

📖 Linux 内核揭秘

分享对 Linux 内核机理的一些浅见，帮助读者理解 Linux 内核机理和其他底层内容。

🗡️pwn.hust.college 教育实践平台🛡️

一个基于神奇宝贝动画设计的趣味教育平台，供学生团体学习和动手实践网络空间安全安全的核心概念。

🎓 Linux 内核高校贡献排行榜

利用高校域名对 Linux 内核开源贡献进行排行，鼓励大家积极向 Linux 内核开展开源贡献。

📦 Mirror CLI

一键换源的命令行工具：支持 Ubuntu，Debian，Deepin，PyPI，crates 等多个发行版及工具

🎓 RISC-V Online

一款基于 WebAssembly 的在线 RISC-V 汇编反汇编工具，方便学习和实践 RISC-V 技术
