华科开放原子开源俱乐部成立于2023年，是在开放原子开源基金会支持下挂牌成立的活跃社区。我们由一群热爱开源文化的师生共同发起，成员来自华中科技大学及全国多所高校，涵盖本、硕、博各阶段。俱乐部遵循开源精神，致力于贡献优秀开源软件、培育开源人才、传播开源文化。

无论你是初探开源的新手，还是经验丰富的开发者，只要你怀有热情并愿意贡献，这里都有你的位置。

## 如何加入我们？

常规招新通常在**每年秋季学期（约9月至11月）** 进行，但我们欢迎在任何时间通过贡献加入。完整的加入流程如下：

### 1. 完成一次开源贡献

我们的核心开源项目均托管在 GitHub：[https://github.com/hust-open-atom-club](https://github.com/hust-open-atom-club)

我们为新人准备了带有 **"good first issue"** label 的任务，难度适中，是绝佳的起步点。

**贡献流程建议**：在感兴趣的 issues 下留言认领 → 与维护者沟通并开始工作 → 提交 Pull Request (PR) → 等待审核与合并。

> 请注意：直接提交 PR 前，建议先通过 issue 与社区沟通，以确保你的工作方向正确并被接纳。

### 2. 提交成员信息表

在您的 PR 被合并后，请填写**俱乐部成员信息表**。

表格位于我们的[文档仓库](https://github.com/hust-open-atom-club/docs)中，填写后请发送至邮箱：**xihu@hust.edu.cn**。

请在表中准确填写您的 GitHub ID，以便我们发出组织邀请。

### 3. 加入俱乐部组织

审核通过后，我们将根据您信息表中的 GitHub ID，邀请您加入 **"hust-open-atom-club"** GitHub 组织，成为正式成员。

### 4. 加入社区交流（可选但推荐）

为了更好的交流与协作，我们强烈建议您加入我们的线上社区：

- **Discord 服务器**：[https://discord.gg/EMJqcQCCpW](https://discord.gg/EMJqcQCCpW)
- **QQ 群**：请关注俱乐部 GitHub 组织首页来获取相关链接。
